# Welcome!

This C# template lets you get started quickly with a simple one-page playground.

```C# runnable
// { autofold
using System;

class Hello 
{
    static void Main() 
    {
// }
double d = 1.5;
Console.WriteLine(d);   // output: 1.5
Console.WriteLine(++d); // output: 2.5
Console.WriteLine(d);   // output: 2.5
Console.WriteLine("Hello World!");

// { autofold
    }
}
// }
```
